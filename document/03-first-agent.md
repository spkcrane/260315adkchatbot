## 🎯 Objective
สร้าง AI agent แรกด้วย Google ADK (Python) - ใช้งาน framework สำหรับ agent development

---

## 🛠️ Prerequisites

ก่อนเริ่ม ตรวจสอบว่า:
- ✅ Codespace พร้อมใช้งาน
- ✅ Gemini API key กรณีที่ยังไม่มีให้ไปสร้างที่ ([Google AI Studio](https://aistudio.google.com/))
- ✅ Python 3.10+ ทำงานได้

```bash
python3 --version
pip --version
```

---

## 📦 Installation & Project Setup

### 1. Install Google ADK

```bash
pip install google-adk
```

### 2. Create Agent Project

Google ADK มี CLI tool เพื่อสร้าง project structure:

### สร้าง folder workshop
```bash
mkdir workshop
cd workshop
```

### create agent ผ่าน Google ADK CLI
```bash
adk create sales_agent
```

Project structure ภายใน folder workshop จะเป็นแบบนี้:
```
sales_agent/
├── agent.py           # Main agent code
├── __init__.py
└── .env              # Environment variables (API keys)
```

---

### Running Your First Agent

ทดสอบ agent ใน 2 วิธี:

#### วิธี 1: Command-line Interface

```bash
# รัน agent ผ่าน CLI
adk run sales_agent

# จากนั้น พิมพ์คำถามได้เลย
```

#### วิธี 2: Web Interface (Recommended)

```bash
# รัน web interface บน localhost:8000
adk web --port 8000

# เปิด browser ไปที่ http://localhost:8000
```

## 🤖 First Agent: Basic Structure

### Default Generated Agent (agent.py)

Google ADK สร้าง agent ดังนี้:

```python
# agent.py - generated by adk create
from google.adk.agents.llm_agent import Agent

# สร้าง root agent - element ที่จำเป็น
root_agent = Agent(
    model='gemini-2.5-flash',
    name='SalesAgent',
    description='A helpful sales assistant for automotive dealership',
    instruction='You are a helpful sales assistant for cars. Be professional and assist customers with their inquiries.'
)
```

## 🎯 Automotive Sales Agent Example

ลองกำหนดหน้าที่ให้ Agent ตอบคำถามเกี่ยวกับเรื่อง Automotive:

```python
# agent.py
from google.adk.agents.llm_agent import Agent

# สร้าง sales agent สำหรับ dealership
root_agent = Agent(
    model='gemini-2.5-flash',
    name='AutoSalesAgent',
    description='Sales assistant for automotive dealership in Thailand',
    instruction="""
Role:
คุณคือที่ปรึกษาการขายของศูนย์รถยนต์

หน้าที่ของคุณคือช่วยลูกค้าเลือกและแนะนำรถยนต์ที่เหมาะสม
โดยอ้างอิงจากความต้องการของลูกค้า เช่น งบประมาณ ประเภทการใช้งาน
จำนวนผู้โดยสาร หรือสไตล์การขับขี่

Goal:
เป้าหมายของคุณคือช่วยลูกค้าค้นหารถที่เหมาะสมที่สุด
โดยอธิบายจุดเด่นของรถแต่ละรุ่นให้เข้าใจง่าย

คุณควรถามคำถามเพิ่มเติมหากข้อมูลของลูกค้าไม่เพียงพอ
เช่น งบประมาณ ลักษณะการใช้งาน หรือความต้องการพิเศษ


Available Vehicles:
- Toyota Camry Hybrid: ฿1,299,000 (sedan, fuel-efficient)
- Honda Civic Type R: ฿2,499,000 (sports car, high performance)
- Toyota Vios: ฿599,000 (city car, economical)


Response:
ตอบลูกค้าในภาษาที่เป็นธรรมชาติ เป็นมิตร และเข้าใจง่าย

เมื่อแนะนำรถ ควรอธิบาย:
- รุ่นรถ
- ราคา
- จุดเด่นของรถ
- เหตุผลว่าทำไมรถรุ่นนี้จึงเหมาะกับลูกค้า

หากมีรถหลายรุ่นที่เหมาะสม
ให้แสดงรายการในรูปแบบ bullet list เพื่อให้อ่านง่าย

ตัวอย่างการตอบ:
รถที่อาจเหมาะกับความต้องการของคุณมีดังนี้:

• Toyota Vios — ราคา ฿599,000  
รถขนาดเล็ก เหมาะสำหรับการใช้งานในเมือง ประหยัดน้ำมัน และราคาประหยัด

• Toyota Camry Hybrid — ราคา ฿1,299,000  
ซีดานขนาดกลาง ขับสบาย ประหยัดน้ำมันด้วยระบบ Hybrid เหมาะสำหรับการใช้งานระยะยาว

Guardrails:
1. ให้แนะนำเฉพาะรถที่มีอยู่ในรายการ Available Vehicles เท่านั้น
2. ห้ามสร้างรุ่นรถใหม่หรือข้อมูลรถที่ไม่มีอยู่จริง
3. หากลูกค้าถามถึงรถที่ไม่มีในรายการ
   ให้แจ้งลูกค้าอย่างสุภาพว่าศูนย์บริการมีเฉพาะรุ่นที่ระบุไว้
4. หากข้อมูลของลูกค้ายังไม่เพียงพอในการแนะนำรถ
   ให้ถามคำถามเพิ่มเติมก่อน เช่น งบประมาณ หรือรูปแบบการใช้งาน
5. คำตอบควรสุภาพ เป็นมิตร และช่วยให้ลูกค้าตัดสินใจได้ง่าย
""",
)
```
